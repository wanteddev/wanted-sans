## 라이선스

Wanted Sans는 [SIL 오픈 폰트 라이선스](https://scripts.sil.org/OFL)로, 글꼴 단독 판매 및 글꼴 라이선스 변경을 제외한 모든 상업적 행위 및 수정, 재배포가 가능합니다.
